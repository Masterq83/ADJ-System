# ADJ-System (Public Research Repository)

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.21205393.svg)](https://doi.org/10.5281/zenodo.21205393)

**Advocatus Diaboli Judgment** — epistemisches Falsifikationssystem für KI-generierte Abweichungen (Lücke Z).

> **Hinweis:** Dieses Repository enthält **Preprint, Benchmark-Ergebnisse und Human-Eval-Material** — **keine** Implementierung des ADJ-Kerns (Betriebsgeheimnis / proprietary) und **keine** PPTX-Präsentation (Entwurf nur lokal). Muster analog zu [metis-philosophy](https://github.com/Masterq83/metis-philosophy) (AGI Thermalistik).
